# On A Winter's Night, A Gift Giver (Unity)

## Overview

On A Winter's Night, A Gift Giver is a short narrative game created for the Winter Cozy Game Jam.

The game explores memory and emotion as the player reflects on past moments to choose a meaningful birthday gift for their father.

Built within a 2-day timeframe, the project focuses on atmosphere, storytelling, and visual presentation.

The concept was partially inspired by creating a meaningful experience for my own dad’s birthday, shaping the game’s focus on memory, reflection, and personal connection.

---

## Features

* Narrative-driven gameplay focused on memory exploration
* Interactive points of interest
* Original pixel art and character design
* Custom animations and visual assets
* Integrated soundtrack
